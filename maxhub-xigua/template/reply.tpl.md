# 🍉 西瓜视频数据采集 回复模板

## 数

### 获取单个作品数据/Get single video data

`GET /api/v1/xigua/app/v2/fetch_one_video`
必填参数：item_id

**返回数据展示：**

| 字段 | 说明 |
|:---|:---|
| ID | 内容唯一标识 |
| 标题 | 内容标题/描述 |
| 作者 | 创作者昵称 |
| 播放量 | 播放/浏览次数 |
| 点赞数 | 点赞/喜欢次数 |
| 评论数 | 评论条数 |
| 分享数 | 分享次数 |
| 发布时间 | 内容创建时间 |
