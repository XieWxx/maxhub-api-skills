# 🦐 皮皮虾数据采集 回复模板

## PiPiXia App


### 搜索接口/Search API

`GET /api/v1/pipixia/app/fetch_search`

**返回数据展示：**

| 字段 | 说明 |
|:---|:---|
| ID | 内容唯一标识 |
| 标题 | 内容标题/描述 |
| 作者 | 创作者昵称 |
| 播放量 | 播放/浏览次数 |
| 点赞数 | 点赞/喜欢次数 |
| 评论数 | 评论条数 |
| 发布时间 | 内容创建时间 |
| 数据说明 | 搜索结果 |
