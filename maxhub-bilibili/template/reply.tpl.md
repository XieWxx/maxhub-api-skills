# 📺 B站数据采集与分析 回复模板

## Bilibili Web

### 提取用户ID/Extract user ID

`GET /api/v1/bilibili/web/fetch_get_user_id`

**返回数据展示：**

| 字段 | 说明 |
|:---|:---|
| ID | 内容唯一标识 |
| 标题 | 内容标题/描述 |
| 作者 | 创作者昵称 |
| 播放量 | 播放/浏览次数 |
| 点赞数 | 点赞/喜欢次数 |
| 评论数 | 评论条数 |
| 发布时间 | 内容创建时间 |
| 数据说明 | 用户ID |
